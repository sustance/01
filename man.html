Usage: omxplayer [OPTIONS] [FILE]
-h / --help              print this help
-n / --aidx  index       audio stream index:eg 1
-o / --adev  device      audio out device: hdmi/local
-i / --info              dump stream format and exit
-s / --stats             pts and buffer stats
-p / --passthrough       audio passthrough
-d / --deinterlace       deinterlacing
-w / --hw                hw audio decoding
-3 / --3d mode           switch tv 3d mode (eg SBS/TB)
-y / --hdmiclocksync     refresh rate match video(def)
-z / --nohdmiclocksync   not refresh rate match video
-t / --sid index         show subtitle with index
-r / --refresh           adjust framerate/res to video
-l / --pos               start position (in seconds)
     --boost-on-downmix  boost volume when downmixing
     --subtitles path    external subs in UTF8 srt fmt
     --font path         subtitle font
     --font-size size    000's screen height(def 55)
     --align left/center subtitle alignment (def left)
     --lines n           in the subtitle buffer(def 3)

z           Show Info
1           Increase Speed
2           Decrease Speed
j           Previous Audio stream
k           Next Audio stream
i           Previous Chapter
o           Next Chapter
n           Previous Subtitle stream
m           Next Subtitle stream
s           Toggle subtitles
d           Subtitle delay -250 ms
f           Subtitle delay +250 ms
q           Exit OMXPlayer
Space or p  Pause/Resume
-           Decrease Volume
+           Increase Volume
Left Arrow  Seek -30
Right Arrow Seek +30
Down Arrow  Seek -600
Up Arrow    Seek +600
